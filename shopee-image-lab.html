ทดสอบดึงรูป Shopee

รองรับทั้ง short link และลิงก์สินค้าปกติของ Shopee
ใช้งานหน้านี้โดยวางลิงก์ Shopee ในช่องด้านบน แล้วกด Fetch Image เท่านั้น ไม่ต้องกดเปิดลิงก์ Shopee ใน browser โดยตรง เพราะ console ของ Shopee จะมี warning/error ของระบบ Shopee เอง เช่น tracking blocked, manifest 404, หรือ 405 ซึ่งไม่ใช่ error ของ lab นี้
โหมดนี้ใช้ endpoint เดิม /api/scrape-product เพื่อยืนยันข้อจำกัดของ Shopee ก่อนใช้ provider จริง
Custom endpoint ควรรับ JSON body: {"url":"..."} และคืน JSON ที่มี image/imageUrl/images/mainImageUrl อย่างใดอย่างหนึ่ง

ไฟล์นี้เป็นหน้าทดลองแยก ไม่เกี่ยวกับปุ่ม Post หรือ webhook ของหน้าหลัก
